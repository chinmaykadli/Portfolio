Connect Umami Cloud (recommended)

The data above is from local browser tracking only. For real visitor analytics across all devices, connect Umami Cloud (free, privacy-friendly, no cookie banner needed).

  1. Sign up at cloud.umami.is (free tier: 10K events/month)
  2. Add your site and copy the website ID
  3. Add this script to all pages before </head>:
    <script defer src="https://cloud.umami.is/script.js" data-website-id="YOUR_ID"></script>
  4. Update the CSP in _headers to allow https://cloud.umami.is in script-src
  5. All events from tracking.js will automatically flow to Umami
